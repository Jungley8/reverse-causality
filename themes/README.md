# 主题系统使用指南

## 生成主题文件

**重要：首次使用前必须生成主题文件！**

### 方法1：使用编辑器脚本（推荐）

1. 在 Godot 编辑器中打开项目
2. 点击菜单：**工具 > 执行脚本**
3. 选择文件：`res://themes/GenerateTheme.gd`
4. 点击"运行"
5. 查看输出面板，应该看到："主题已保存到 res://themes/default_theme.tres"

### 方法2：手动运行（如果方法1失败）

如果编辑器脚本无法运行，可以：

1. 在 Godot 编辑器中打开 `themes/GenerateTheme.gd`
2. 在脚本编辑器中右键点击
3. 选择"运行脚本"（如果有此选项）

## 验证主题已生成

生成成功后，你应该能在 `themes/` 目录下看到 `default_theme.tres` 文件。

## 主题特性

新的主题系统包含：

- ✅ **更明显的按钮样式**：更大的圆角（12px）、更粗的边框（2-3px）、明显的阴影
- ✅ **增强的卡片效果**：10px圆角、2-3px边框、光晕效果
- ✅ **改进的槽位视觉**：不同状态有不同颜色和光晕（绿色=有效，红色=无效）
- ✅ **更大的字体**：按钮字体从16px增加到18px
- ✅ **更明显的阴影**：增强的阴影效果，使UI元素更有层次感
- ✅ **统一的配色方案**：基于 m3.md 规范的完整配色系统

## 如果UI没有变化

如果运行游戏后UI没有明显变化，请检查：

1. ✅ 主题文件是否已生成（`themes/default_theme.tres` 是否存在）
2. ✅ 在 Godot 编辑器中：**项目 > 重新扫描项目文件**
3. ✅ 重新运行游戏
4. ✅ 检查控制台输出，应该看到："✅ 主题已加载：res://themes/default_theme.tres"

## 重新生成主题

如果需要修改主题样式：

1. 编辑 `themes/GenerateTheme.gd` 中的颜色和样式值
2. 重新运行编辑器脚本生成新的主题文件
3. 重新扫描项目文件
4. 重新运行游戏
